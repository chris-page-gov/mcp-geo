# LandIS Licence and Open Access Status

Last reviewed: 2026-04-04

## Current position captured for the MVP

- The LandIS home page stated that LandIS soils had become open access and
  linked to `https://portal.landis.org.uk/`.
- The detailed portal licence and redistribution terms were not fully extracted
  into this repo's research environment on 2026-04-04.
- The MVP therefore treats licence status as improved access with remaining
  validation requirements, not as blanket permission for unrestricted
  redistribution.

## Safe operating rule for MCP outputs

- Return provenance and source links on every LandIS-derived response.
- Keep checked-in documentation summaries separate from any future bulk-data
  exports.
- Revalidate the current portal licence before publishing new downstream data
  packages or derivative datasets outside the repo/runtime environment.
