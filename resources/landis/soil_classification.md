# LandIS Soil Classification Guidance

This MCP resource is a concise operational summary of the LandIS national soil
map and soil-classification guidance for non-specialist callers.

## What to communicate in tool outputs

- Soilscapes classes are generalized landscape-scale classes intended for
  screening and explanation, not parcel-scale certainty.
- Association and series terminology should be preserved in outputs when the
  source layer is NATMAP or a join-expanded derivative.
- Plain-language summaries should keep the original class labels available so a
  specialist can trace back to the source classification.

## Minimum caveat set

- Generalized mapping can hide local artificial ground, recent disturbance, and
  fine-scale variation.
- A conversational or report output is not a substitute for field
  investigation, geotechnical testing, or regulatory advice.
