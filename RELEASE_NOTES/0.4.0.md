# Release 0.4.0

Date: 2026-02-25

## Summary

`0.4.0` packages the latest reliability and reporting work for `mcp-geo`,
including a Long Horizon-style Codex session summary workflow and a new
repository extent/complexity analysis skill with a manager-facing report card.

## Highlights

- Added Long Horizon-style Codex reporting for repo-scoped sessions:
  - `scripts/codex_long_horizon_summary.py`
  - skill package `skills/mcp-geo-long-horizon-summary/`
  - deterministic summary SVG template and image-first markdown report output
  - baseline artifacts under `docs/reports/mcp_geo_codex_long_horizon_summary_2026-02-25.{md,json,svg}`
- Added repository extent/complexity analysis capability:
  - `scripts/repo_extent_complexity_report.py`
  - skill package `skills/mcp-geo-repo-extent-complexity/`
  - dual-scope measurement (`git_tracked`, `workspace`), generated/output exclusion,
    Python cyclomatic complexity, churn-weighted hotspots, optional GitHub stats
  - baseline artifacts under `docs/reports/repo_extent_complexity_2026-02-25.{md,json}`
- Added manager-facing report card output for non-technical stakeholders:
  - `docs/reports/repo_extent_complexity_report_card_2026-02-25.md`
  - plain-English terminology, metric basis/sources, and practical implications
- Included regression coverage for new reporting workflows:
  - `tests/test_codex_long_horizon_summary.py`
  - `tests/test_repo_extent_complexity_report.py`
- Updated full-suite regression assertions for current tool contracts:
  - `tests/test_evaluation_harness_full.py`
  - `tests/test_golden_scenarios.py`
  - `tests/test_tool_upstream_endpoint_contracts.py`
