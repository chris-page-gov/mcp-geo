# Release 0.2.7 - 2026-01-29

Focus: map handling hardening, sandbox safety, and map UI stability.

## Highlights
- OSM proxy now caches tiles and supports configurable base URL + contactable user agent.
- Geography selector stability improvements: async overlay queue, batched admin lookups, centralized diagnostics.
- MapLibre CSP worker is now served locally to reduce external dependencies.

## Added
- Playground debug tab with runtime snapshot, HMR status, and redacted logs.
- MCP prompts list/get backed by evaluation prompt examples.
- Geography selector diagnostics panels now include source/render counts and in-view counts.
- Map handling review report at `docs/map_handling_review.md`.
- Map proxy unit coverage and a geography selector style-switch Playwright flow.

## Changed
- OSM tile proxy adds caching headers and configurable base/UA/contact settings.
- Playground sandbox requires explicit allow-same-origin opt-in outside dev mode.
- Geography selector map lifecycle now uses a MapLibre adapter and async mutation queue.
- Geography selector tool calls are routed through places/admin lookup service helpers.
- CSP allowlist tightened for geography selector; MapLibre CSP worker served locally.

## Fixed
- Geography selector boundary fallback retries without geometry on 5xx and avoids missing overlays after style reloads.

## Upgrade Notes
- Set `OSM_TILE_USER_AGENT` and `OSM_TILE_CONTACT` if you rely on OSM tiles at scale.
- Rebuild the Docker image after upgrading (`docker build -t mcp-geo-server .`).
