# Release Notes 0.3.0 (2026-02-13)

## Added
- Completed the parallel OS catalog gap workstreams, including OS Downloads, OS Net, NGD OTA discovery, raster/WMTS and WFS capability tools, and QGIS linkage tools.
- Added `os_qgis.vector_tile_profile` and `os_qgis.export_geopackage_descriptor` with `delivery=inline|resource|auto`.
- Added OS cache/export resources for delayed and large-payload delivery (`resource://mcp-geo/os-cache*`, `resource://mcp-geo/os-exports*`).
- Expanded tool discovery/search organization with toolset filters and broader endpoint catalog coverage.

## Changed
- Extended observability with delivery fallback metrics and structured OS Downloads export lifecycle logging.
- Aligned Docker packaging/runtime behavior for this release stream:
  - Included required OS vector style assets from `submodules/os-vector-tile-api-stylesheets` in build context.
  - Removed `EXPOSE 8000` from the default STDIO image configuration.
- Updated docs for playground port behavior (`5173` dev, `4173` Playwright tests).

## Fixed
- Fixed STDIO `resources/read` resolution for `resource://mcp-geo/*` data resources.
- Fixed request validation for `/tools/call` and `/tools/search` to return `400 INVALID_INPUT` on non-object JSON.
- Fixed static map sizing above a single tile to honor requested output dimensions.
- Fixed QGIS export/resource path handling and profile behavior when style submodules are absent.
- Hardened Playwright CDN route stubbing with resilient wildcard matching for `shpjs`.
