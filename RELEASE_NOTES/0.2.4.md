## Upgrade Notes
- Rebuild the Docker image after upgrading (`docker build -t mcp-geo-server .`).
- Use `MCP_STDIO_UI_SUPPORTED=1` to force MCP-Apps UI mode.
- Use `MCP_STDIO_FALLBACK_BBOX_DEG` to tune the fallback map span.
