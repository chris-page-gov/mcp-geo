# Release Notes 0.2.9 (2026-02-01)

## Added
- Boundary cache audit tools and dataset freshness metadata.
- Boundary pipeline helpers: latest report, tracker, status ticker, triage, and auto-fix loop.
- Post-run checklist mapping pipeline validation errors to next actions.

## Changed
- Boundary pipeline now reports download/extract/table sizes and totals them in the tracker.
- Boundary manifest validation overrides updated (NI LGD fallback, TTWA duplicate codes).
- Devcontainer setup now installs boundary ingest dependencies and enables cache env defaults.

## Fixed
- Admin lookup live calls now tolerate per-source failures and return partial results.
- latest_reports helper now provides clearer cache-disabled hints and suppresses noisy warnings.
