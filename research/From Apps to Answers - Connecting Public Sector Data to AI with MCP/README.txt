Prism bootstrap pack: MCP research brief

This folder is a minimal LaTeX project you can upload into Prism, or paste file-by-file.

Files:
- main.tex
- references.bib
- sections/*.tex

Suggested Prism workflow:
1) Create a new Prism project.
2) Upload these files (or copy/paste).
3) Compile main.tex (Prism shows source + preview).
4) Use Prism's built-in AI to:
   - refine tone for non-technical public sector readers
   - add citations + literature links
   - maintain consistency across sections

Definition of Done (for the brief):
- Audience outcomes finalised (3–5)
- 10–20 credible citations added (mix of academic + government/standards)
- Case study appendix written (peatmap + audit evidence summary)
- Slide spine drafted from the outcomes and patterns
