# Notebook Workspace

This directory contains optional notebook-based tracking for map delivery experiments.

- `map_delivery_option_tracker.ipynb`: Loads trial observations and prints a compact run matrix.

Suggested usage (inside the devcontainer):

```bash
jupyter lab --ip=0.0.0.0 --port=8888 --no-browser
```

Then open forwarded port `8888` from your IDE.
