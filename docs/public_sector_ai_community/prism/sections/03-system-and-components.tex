\chapter{System and Components}
\label{ch:system}

\section{Runtime Structure}

The system exposes MCP functionality through:
\begin{itemize}
  \item HTTP transport (including \texttt{/mcp}),
  \item STDIO transport via JSON-RPC adapter,
  \item tool and resource catalogs with discovery and call endpoints.
\end{itemize}

\section{Component Map}

\begin{itemize}
  \item \texttt{server/}: transports, routing, protocol handling
  \item \texttt{tools/}: domain handlers (OS, ONS, NOMIS, admin, map)
  \item \texttt{resources/}: static or generated resource payloads
  \item \texttt{tests/}: regression, compatibility, and harness validation
\end{itemize}

\section{Current Functional Scale}

Current registry evidence shows 81 tools loaded in the runtime registry.
