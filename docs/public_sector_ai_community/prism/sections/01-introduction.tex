\chapter{Introduction}
\label{ch:introduction}

\section{Purpose}

This document package explains what was discovered and built in the \texttt{mcp-geo}
repository for UK public-sector delivery audiences.

\section{Audience}

The text is designed for:
\begin{itemize}
  \item novice readers new to MCP and geospatial AI integration,
  \item apprentices with early-stage coding knowledge,
  \item delivery and assurance reviewers.
\end{itemize}

\section{Scope}

The documented scope includes this repository's implementation, testing,
troubleshooting, and release evidence. It excludes implementation internals of
git submodules.
